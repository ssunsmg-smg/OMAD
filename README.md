# OMAD 마케팅 생성기

한영주류 영업팀 전용 AI 마케팅 콘텐츠 자동 생성 툴

---

## 🔗 접속 URL

```
https://omad-git-main-ssunsmg-s-projects.vercel.app
```

PC / 모바일 모두 접속 가능 (모바일: 우상단 ☰ 메뉴)

---

## ✦ 주요 기능

| 기능 | 설명 |
|------|------|
| AI 상권 자동 분석 | 주소 입력 → 카카오 지도 기반 실제 상권·고객층 분석 |
| 네이버 플레이스 | 최적화 체크리스트 10개 자동 생성 |
| 블로그 | 네이버 블로그 방문후기 초안 (1500자) |
| 인스타그램 | 본문 + 해시태그 15개 |
| 페이스북 | 공유 유도형 게시물 |
| 유튜브 | 제목 후보 3개 + 설명란 + 태그 |
| 티스토리 | SEO 최적화 포스팅 (2000자) |
| 상권 분석 | 고객층·경쟁환경·추천 주류·영업 핵심포인트·방문 적기 |

---

## 🛠 기술 스택

- **프론트엔드**: HTML / CSS / JavaScript (Vanilla)
- **백엔드**: Vercel Serverless Function (`api/gemini.js`)
- **AI**: Claude Sonnet (Anthropic API)
- **지도**: 카카오 Local API (주소→좌표, 주변 상권 데이터)
- **호스팅**: Vercel (GitHub 연동 자동 배포)

## 🚀 업데이트 방법

1. 이 저장소에서 파일 수정 (GitHub 웹 편집기 또는 로컬)
2. Commit → Vercel 자동 재배포 (1~2분 소요)

---

## 📱 모바일 사용법

1. 브라우저에서 URL 접속
2. 우상단 ☰ 버튼 → 거래처 정보 입력
3. **상권·고객층 자동 입력** 버튼으로 AI 분석
4. **✦ 생성하기** 버튼 클릭
5. 각 탭에서 콘텐츠 확인 및 복사

홈 화면에 추가하면 앱처럼 사용 가능 (공유 → 홈 화면에 추가)
